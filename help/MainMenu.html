1. メイン画面の機能について
    1. メニュー
      1. ファイル
        1. 新規作成
          タイムテーブルを新たに作成します。
        2. 開く
          作成済みのタイムテーブルを開きます。
        3. 上書き保存
          現在編集中のタイムテーブルを保存します。
        4. 名前を付けて保存
          現在編集中のタイムテーブルを新しいファイルに保存します。
        5. 印刷
          現在編集中のタイムテーブルを印刷します。
        6. 印刷プレビュー
          印刷プレビューダイアログを表示します。
        7. 最近使用したファイル
          最近使用したファイルの一覧を表示します。
        8. 終了
          このプログラムを終了します。
      2. 編集
        1. 切り取り
          現在選択中の行の切り取りを行います。
        2. コピー
          現在作業中の行をコピーします。
        3. 貼り付け
          切り取りまたはコピーした行を現在の行に貼り付けます。
        4. 行クリア
          現在作業中の行をクリアします。
      3. タイムテーブルの設定
        1. 表示範囲を自動設定
          現在カレンダー部分に表示している期間を自動設定します。（自動設定の詳細）
        2. 選択範囲を自動設定
          現在選択している部分の期間を自動設定します。（自動設定の詳細）
        3. 表示期間
          表示範囲の指定ダイアログを表示します。
        4. タイムテーブルの設定
          タイムテーブルの設定ダイアログを表示します。
      4. ツール
        1. オプション
          オプション設定ダイアログを表示します。
      5. ヘルプ
        1. 内容
        2. インデックス
        3. 検索
        4. バージョン情報
    2. ツールバー
      1. 全般
        1. 新規作成
        2. 開く
        3. 上書き保存
        4. 印刷
        5. 切り取り
        6. コピー
        7. 貼り付け
        8. ヘルプ
      2. その他
        1. 表示範囲を自動設定
        2. 選択範囲を自動設定
        3. 表示期間
        4. 設定
